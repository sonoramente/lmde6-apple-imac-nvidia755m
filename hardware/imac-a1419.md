# Apple iMac A1419 (Late 2013)

## Tested Configuration

- **Model Identifier:** iMac14,2
- **CPU:** Intel Core i5
- **GPU:** NVIDIA GeForce GT 755M (2 GB)
- **Firmware:** Apple EFI (UEFI)
- **Storage:** SATA SSD
- **RAM:** DDR3

## Notes

This system was tested with LMDE 6 using the proprietary NVIDIA driver.
UEFI boot mode was used (legacy BIOS not required).

Thunderbolt display output was verified.
